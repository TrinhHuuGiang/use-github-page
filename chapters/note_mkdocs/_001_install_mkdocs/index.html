Aboud MkDocs

MkDocs is a static site generator geared towards building project documentation. - Documentation source files: Markdown - Configuration by: YAML configuration file.

Refer: - https://www.mkdocs.org/ - https://www.mkdocs.org/user-guide/installation/

Install

Requirements: Python, pip

Install MkDocs: pip install mkdocs
